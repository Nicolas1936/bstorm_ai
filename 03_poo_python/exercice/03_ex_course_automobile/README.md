# Simulation de course automobile

## class Voiture
caractérisé par :
- v_min
- v_max

actions : 
- vitesse de la voiture à chaque tours de manière aléatoire

## class Course
caractérisé par : 
- nb_tour_total

actions : 
- revoie le temps que la voiture a mis pour faire son tour du circuit

## class Circuit
caractérisé par : 
- distance (km)
